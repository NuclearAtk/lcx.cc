常见问题：

如果你无法启动本软件，或者启动报错，我这里之前测试很久，十分稳定，我写的软件，一般不会出现此类情况，原因可能如下：

1、只支持 32 位 Windows 操作系统（64位的听说可以，由于环境有限我无法测试……），Windows XP（XP 以下未测试，这年头应该没人用了吧） 至 Windows 7 旗舰版，新出来的 Windows 8 未测试，理论上应该没有任何问题。

2、VMP 壳的问题，这个主要是防止非主流黑客篡改，本人以人品保证绝无任何木马插件，本人写的软件从来不加这些恶心东西，大家有目共睹的。

如果碰到启动的时候提示：文件访问错误或其它错误，那么可能原因如下：

因为启动的时候要释放几个控件到系统目录（微软官方控件，无毒），所以 Win7、Win8 默认估计无权操作，需要使用管理员权限运行。

在使用 Crazy Scan 扫描的时候，如果碰到掉线，掉线是因为发包太快（但是这不影响准确率，出错会自动重试，绝不漏过任何一条路径），请你尝试减少连接数再试。

被杀的问题，原因可能是 VMP 壳被杀，至于被 360 杀嘛，我就不解释了，你懂的……

至于扫描速度嘛，从大家的反馈来看，相当的给力……
