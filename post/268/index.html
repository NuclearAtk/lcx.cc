第三方地图使用方法：

1、从指定网址下载地图。

2、下载完后，如果是压缩包，则解压。

3、解压完会出现一个vpk格式的文件，或者是一个文件夹，文件夹中有一个vpk格式的文件。

4、将这个vpk文件（不要文件夹），单独放入游戏附件文件夹中，路径类似这样的：游戏主目录路径\left4dead2\addons\，例如我的路径为：E:\单机游戏\求生之路2 2077版\left4dead2\addons\，这个文件夹路径每个人都不同，请根据你的游戏安装路径自行判断。

5、完成后的路径，应该类似这样的：E:\单机游戏\求生之路2 2077版\left4dead2\addons\文件名.vpk。

6、部分地图可能自带模型之类的，会和其他地图冲突，不想玩了，就直接删除这个vpk文件即可，很简单。

第三方地图建图指令的寻找方法：

求生之路的命令很简单，不过还是有小白不知道，这里我就讲一下建图指令的寻找方法。

首先打开控制台，什么是控制台？控制台是游戏的开发人员用来调试游戏一种命令行控制系统，简称控制台，你只需要简单的理解为，他可以执行各种游戏指令就行了，当然，也包括各种的作弊指令……

什么是游戏指令？求生之路2秘籍及作弊码和控制台全套指令：https://lcx.cc/post/182/，这些全都是游戏指令，这里只收录了很少一部分，游戏指令有相当多，常用的就只有这些。

打开控制台方法，按 ~ 这个键，就是ESC下边，数字1左边，别跟我说你找不到！

按下以后，会弹出来控制台，如果没有弹出来，说明，你没有启用控制台，启用方法：

游戏主界面，选项-->键盘/鼠标-->允许使用开发者控制台-->启用

在弹出的命令行中输入：map，然后按空格，会出来很多地图列表，按上下键可以选择，求生之路2 2091版官方完整地图（按剧情顺序排列）：https://lcx.cc/post/1946/，除了这些以外，就是第三方地图了，一般第三方地图指令和他的名字非常相似，例如死亡之船2 (Death Aboard II)：

l4d_deathaboard01_prison

格式一般为：

l4d_地图名关卡数_关卡名字

地图名关卡数_关卡名字

地图名关卡数

地图名

非常好找，有的地图不带关卡号，那就任意进一关，然后选择重新开始战役，就跳到第一关了，然后打开控制台，输入：status，会显示游戏状态，在map 那一行中，就是当前地图名，例如：

hostname: QQ Qun 162033511
version : 2.0.7.7 4632 insecure  (unknown)
udp/ip  : 119.57.51.253:27015 [ public same ]
os      : Windows Dedicated
map     : c2m2_fairgrounds
players : 16 humans, 0 bots (20 max) (not hibernating) (unreserved)

这个要自己练习，多多实践，就很容易找到地图指令了。

建立地图，输入：map 你找到的地图名，然后按回车，就开始游戏了，游戏通用的建立地图命令。
