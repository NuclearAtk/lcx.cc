ckeditor 精简与详细配置方法

2011-11-28
老系统内容
约 5073 字 预计阅读 11 分钟

文章目录

一.ckeditor 3.0.1精简

精简前:4.52M 精简后:853K
1.删除_samples和_source文件夹,分别为示例文件和未压缩源程序
2.删除lang文件夹下除zh-cn.js,en.js下的所有语言文件.根据需要删除
3.删除根目录下的changes.html(更新列表),install.html(安装指向),license.html(使用许可).
4.删除skins目录下不需要的皮肤.我一般用V2(简单.朴素) //如果只保留V2则必须在config.js中指定皮肤

二.ckeditor 3.0.1相关文件配置路径
1./ckeditor.js 核心文件,调用需加载
2./config.js 配置文件,参数配置均在此完成
3./plugins/smiley/images 表情符号.

三.ckeditor应用

方法1，引用js脚本，然后将Class的值设置为“ckeditor”：
<script type="text/javascript" src="ckeditor/ckeditor.js"></script>
<textarea id="editor1" name="editor1" rows="10" cols="80" class="ckeditor">初始化html内容</textarea>

方法2，首先在页面里加载核心js文件，接着按常规方式放置textarea，然后在textarea后面写js替换textarea，其他调用方式可参考 _samples 目录下的示例。
<script type="text/javascript" src="ckeditor/ckeditor.js"></script>
<textarea id="editor1" name="editor1" rows="10" cols="80">初始化html内容</textarea>
<script type="text/javascript">CKEDITOR.replace('editor1');</script>

插入图片

CKEDITOR.instances.content.insertHtml(“<img src=…>”);

设置值、取值

设置值

CKEDITOR.instances.content.setData(“千一网络”); // content 就是前面 CKEDITOR.replace 的第一个参数值

或

var editor = CKEDITOR.replace(“content”);
editor.setData(“千一网络”);
